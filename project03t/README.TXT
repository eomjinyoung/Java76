[주제: Front Controller 패턴 적용]
1. 프론트 컨트롤러 클래스 작성
  - DispatcherServlet.java 클래스 작성
2. 프론트 컨트롤러와 페이지 컨트롤러의 호출 규칙 정의 
  - PageController.java 인터페이스 작성
3. Error.jsp 작성
4. 기존의 서블릿 클래스를 페이지 컨트롤러로 전환하기
  - BoardListServlet ---> BoardListController 
5. 프론트 컨트롤러 서블릿을 web.xml에 등록한다.
  - 페이지 컨트롤러로 전환된 서블릿은 제거한다.
