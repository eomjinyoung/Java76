[HTML(Hyper-Text Markup Language) - 1단계]
1. 기본 구조
=> test01.html (HTML 4.01)
=> test02.html (HTML 5)
  - 기존의 HTML 4.01 보다 태그 문법이 간결해졌다.
  - 웹브라우저 마다 HTML5 문법을 지원하는 범위가 다르다.
    => HTML5에 정의된 태그를 사용할 때 어느 브라우저에서 지원하는 지 살펴야 한다.

2. 텍스트 표현하는 태그
1) h1,h2,h3,h4,h5,h6 - 제목을 표현하는 태그
=> test03.html
=> test04.html : 시맨틱 웹의 의미 

2) p - 문단을 표현하는 태그, br - 강제 줄 바꿈 태그
   pre - 콘텐츠 내부의 줄 바꿈 기호를 무시하지 않고 처리한다. 
=> test05.html

3) b, i, strong - 키워드를 강조하는 태그
   address - 웹 사이트에 대한 간략한 정보나 권리 등을 표현하는 태그
=> test06.html

4) a - 외부에 연결된 문서를 표현하는 태그
=> test07.html



3. 구조화된 데이터를 표현하는 태그
1) table, tr, th, td - 데이터를 테이블로 표현하는 태그
=> test08.html
=> test09.html : 셀 병합

2) ol, ul, li - 데이터를 목록으로 표현하는 태그
=> test10.html

4. 서버 애플리케이션과 연동하는 태그
1) form, input, textarea, select, button - 입력폼을 만들 때 사용하는 태그 

5. 그룹으로 묶는 태그
1) div - UI 컴포넌트를 묶을 때 사용하는 태그
2) span - 특정 데이터를 분류할 때 사용하는 태그
