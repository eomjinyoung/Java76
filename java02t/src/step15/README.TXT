[주제: 예외 처리]
문법의 목적:
1) 실행 중에 오류가 발생했을 때 시스템을 멈추지 않고 계속 실행하게 만드는 문법!
   실행 중에 오류가 발생하면, 적절한 조치를 취한 후 실행을 계속하게 만든다.
2) 작업 코드와 오류 처리 코드를 구분함으로써 코드의 가독성을 높이는 문법!
3) 오류 내용을 상위 호출자에게 효과적으로 전달하는 문법!

ex01 - 예외 던지고 받기
  => 예외 던지는 기본 문법
  => 예외를 받는 기본 문법
  => 예외 처리를 상위 호출자에게 위임하는 방법

ex02 - Error와 Exception 예외 : 시스템 예외와 애플리케이션 예외 차이점  
  => 예외를 최상위 호출자까지 조용히 전달하는 방법 
  => Error 계열 - 시스템 예외
  => Exception 계열 - 애플리케이션 예외
     => RuntimeException - 스탤스 모드

ex03 - finally 블록!
  
ex04 - 사용자 정의 예외!
  => Exception 또는 RuntimeException의 하위 클래스를 정의한다.
  => 특별히 기능을 추가한다기 보다는 예외 이름을 정의하기 위해서 사용한다.
